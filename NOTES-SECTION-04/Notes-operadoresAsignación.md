### Operadores de asignación
Antes de realizar cualquien asignación primero se debe declarar la variable a la cual se dara esa misma.

```
Suma de asignación
a = 0
a += 1          #1

Resta de asignación
b = 2
b -= 1          #1

Multiplicación
c = 9
c *= 3          #27

División
d = 8
d /= 2          #4

Módulo de asignación
e = 5
e %= 2          #1

Exponente
f = 5
f **= 2         #25
```
